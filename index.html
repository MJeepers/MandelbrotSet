The Mandelbrot Set

Understanding the Mandelbrot Set

What is it? The Mandelbrot Set is a famous set of complex numbers that creates a highly intricate two-dimensional fractal shape. It is defined by the set of complex numbers "c" for which the function f(z) = z2 + c does not diverge when iterated from z = 0.

Mathematics & Fractals: As a fractal, the Mandelbrot Set is "self-similar," meaning you can zoom in infinitely and continue to find complex patterns that resemble the whole structure. It serves as a cornerstone in the study of chaos theory and complex dynamics.

Artistic Significance: Beyond math, it is celebrated as "Art by Equation." Because the boundary of the set is infinitely complex, it produces stunning, psychedelic visualizations where colors are often assigned based on how quickly the numbers escape toward infinity.

How to Use the Simulation

This interactive simulation renders the Mandelbrot Set in real-time using your browser's hardware. To explore the infinite depth of the set:

  • Zoom In: Click anywhere on the fractal to zoom into that specific coordinate.
  • Calculation: Every time you click, the engine recalculates thousands of points using the formula zn+1 = zn2 + c.
  • Detail: As you zoom deeper, the simulation automatically increases its iteration count to maintain visual clarity.
